# Sales Performance Excel Dashboard

This project is an interactive **Excel sales analytics dashboard** designed to analyze and visualize sales performance across different regions and time periods.

The dashboard compares **Current Year vs Previous Year sales**, monitors performance against targets, and allows users to filter results by geographic region.

## Dashboard Features

- Monthly sales comparison (Current Year vs Previous Year)
- Target tracking with 5% growth benchmark
- Interactive filters for:
  - Continent
  - Country
  - Returned / Sold products
- Sales breakdown by order date
- Dynamic charts for performance monitoring
- Pivot tables and slicers for interactive analysis

## Key Insights

The dashboard enables businesses to:

- Identify top performing months
- Compare historical sales trends
- Monitor performance against sales targets
- Analyze regional sales performance
- Support data-driven decision making
